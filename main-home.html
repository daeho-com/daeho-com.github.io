코드에 쓴 태그에 대해 알려줘!!

nav, div, a 태그에 대해 설명해줄게!!

nav 요소는 사이트 내비게이션 구역을 감싸는 태그야.

메뉴 링크들을 묶어서 보여줘!


div 태그는 아무런 의미 없는 블록 컨테이너야.

레이아웃 구분이나 스타일 적용용으로 사용해.


a 태그는 하이퍼링크를 만드는 태그야.

href 속성에 URL을 지정하면 클릭 시 해당 페이지로 이동해.

직접 실습해볼래!!

      <a class="top_nav-login" href="login.html">로그인</a>
    </div>
  </nav>
